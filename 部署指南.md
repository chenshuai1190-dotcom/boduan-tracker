# 波段跟踪计划 2.0 - 部署指南 

## 你即将做什么

把这个文件夹里的代码上传到 Vercel,自动变成一个网站,然后加到 iPhone 主屏幕变成 App。

---

## 第 1 步:在 Vercel 创建新项目

1. 打开 https://vercel.com,登录你的账号
2. 看到主面板后,点右上角黑色的 **「+ Add New」** 按钮
3. 选择 **「Project」**
4. 你会看到 "Import Git Repository" 页面
5. **不要导入 Git**,在页面下方找到 **「Clone Template」** 或 **「Deploy a template」** 区域
6. **直接关闭这个页面**,我们用更简单的方式

### ✨ 简单方式:用 Vercel CLI 拖拽部署(推荐)

实际上 Vercel 网页版部署必须连 GitHub。但我们有 **Plan B**:

**用 vercel.com/new 的拖拽功能**

1. 打开 https://vercel.com/new
2. 这个页面顶部有个区域可以拖拽 zip 文件
3. **如果看不到拖拽区域**,说明你的 Vercel 账号不支持拖拽,需要走下面的 GitHub 路线

---

## 第 1B 步:如果必须用 GitHub(实际情况)

⚠️ 实话:Vercel 网页版部署**强制要求 GitHub**。但我们可以**不用 git 命令行**,用 GitHub 网页版上传:

### 在 GitHub 网页创建仓库

1. 打开 https://github.com/new
2. **Repository name** 填:`boduan-tracker`
3. **Public** 或 **Private** 都可以(选 Private 更安全)
4. **不要勾选** Add README、.gitignore、license(全部不勾)
5. 点绿色的 **「Create repository」** 按钮

### 网页上传文件

1. 创建后会进入空仓库页面
2. 找到一个链接 **「uploading an existing file」**(在 quick setup 区域),点击
3. 把 **boduan-tracker 文件夹里所有的文件**(不是文件夹本身,是里面的内容)拖拽到上传区域
   - 注意:要包含 `package.json`、`vite.config.js`、`index.html`、`src/`、`api/`、`public/` 等所有内容
4. 等所有文件上传完成(显示绿色对勾)
5. 在下方的 commit message 填:`first commit`
6. 点绿色的 **「Commit changes」** 按钮

### 在 Vercel 导入这个仓库

1. 打开 https://vercel.com/new
2. 找到你刚创建的 `boduan-tracker` 仓库,点旁边的 **「Import」**
3. **关键设置**:
   - **Framework Preset**:选 **Vite**(应该会自动识别)
   - **Build Command**:保持默认 `vite build` 或 `npm run build`
   - **Output Directory**:保持默认 `dist`
4. 在 **Environment Variables**(环境变量)区域:
   - **Name** 填:`FINNHUB_API_KEY`
   - **Value** 填:你的 Finnhub API Key(那串 `ctkj...` 字符)
   - 点 **「Add」** 添加
5. 点最下方的蓝色 **「Deploy」** 按钮
6. 等 1-3 分钟,看到 🎉 烟花动画就成功了
7. 点 **「Visit」** 按钮,看到你的 App!

---

## 第 2 步:加到 iPhone 主屏幕(变成 App)

1. 在 iPhone Safari 打开你的 Vercel 网址(类似 `boduan-tracker-xxx.vercel.app`)
2. 点底部中间的 **分享按钮**(方框 + 向上箭头)
3. 在弹出菜单往下滑,找到 **「添加到主屏幕」**
4. 修改名字(可选),点右上角 **「添加」**
5. 退出 Safari,在主屏幕找到一个新图标(深蓝色 + BD 字样)
6. 点击它,就像打开真正的 App 一样!

## 第 3 步:测试

打开你的 App 后:

1. 滚到底部,看到蓝紫渐变的 **「拉取实时行情」** 按钮

## 常见问题

**Q1:点拉取行情后报错 "API key 未配置"**
A:你忘了在 Vercel 设环境变量。回到 Vercel 项目设置 → Environment Variables → 添加 `FINNHUB_API_KEY`,然后重新部署一次(在 Deployments 选项卡点 ⋯ → Redeploy)。

**Q2:股价显示 0 或不准确**
A:Finnhub 免费版美股**只在交易时段准确**(美东时间 9:30-16:00,北京时间约晚上 9:30 - 凌晨 4:00)。盘后请手动输入。

**Q3:VIX 显示不出来**
A:Finnhub 免费版可能不支持 VIX(`^VIX`)。如果拉不到,VIX 需要手动输入。可以去 https://www.cnbc.com/quotes/.VIX 查实时值。

**Q4:每天 60 次免费调用够用吗?**
A:Finnhub 免费版是**每分钟 60 次**,不是每天。完全够用。

**Q5:数据会不会丢?**
A:数据存在你手机浏览器里,只要不清缓存就在。建议每次大改后点「保存」按钮。

**Q6:能否多设备同步?**
A:这个版本不行(数据本地存储)。下一版可以加云同步,需要用 Vercel KV 或 Supabase。
